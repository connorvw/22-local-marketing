# 22 Local Marketing Launch Checklist

## Identity

- Use the final orange-square logo, not the map-pin exploration.
- Export favicon and social avatar from the logo kit.
- Use the full wordmark on the website header, proposal covers, and sales collateral.
- Use the square mark alone only in tiny or icon-style placements.

## Website

- Install Sora.
- Add the brand token CSS variables.
- Build the homepage around clear outcomes: calls, rankings, booked jobs, service areas.
- Include one strong CTA above the fold.
- Add service pages for website builds, local SEO, and paid acquisition.
- Add tracking language and proof points near service claims.

## Social

- Default feed posts to 1080 x 1080.
- Use one of the approved Blaze frame treatments on every post.
- Keep the logo small.
- Use one idea per post.
- Create reusable templates for insights, KPIs, service promos, testimonials, and case studies.

## Ads

- Build square, story, and landscape variants for each campaign.
- Keep each ad to one headline, one support line, and one CTA.
- Use specific local outcomes instead of broad growth promises.
- Avoid over-designed graphics and stock agency imagery.

## Collateral

- Use the proposal cover template for new-business documents.
- Use the service sheet template for one-page sales handouts.
- Use the report card template for monthly client updates.

## Voice

- Replace vague marketing language with operational language.
- Mention service areas, calls, reviews, rankings, conversion, and revenue.
- Keep CTAs direct.

## Quality Control

- Check contrast before publishing.
- Check logo clear space.
- Check mobile crop on social posts.
- Check that all headlines fit without touching the frame.
- Check that the orange accent is present but not overwhelming.
