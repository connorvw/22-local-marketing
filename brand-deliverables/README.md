# 22 Local Marketing Brand Deliverables

This folder contains the supporting brand-system deliverables for 22 Local Marketing.

## Files

- `voice-guide.md`: tone, messaging, CTAs, headline examples, words to use and avoid.
- `logo-usage-guide.md`: logo rules, clear space, minimum size, approved usage, incorrect usage.
- `photography-art-direction.md`: image style, subjects, treatments, and what to avoid.
- `ad-creative-guidelines.md`: ad formats, copy frameworks, layout rules, creative examples.
- `website-system.md`: website structure, component guidance, page patterns, mobile rules.
- `collateral-templates.html`: visual templates for proposal cover, service sheet, and report card.
- `brand-tokens.css`: implementation-ready CSS tokens and small brand utility classes.
- `launch-checklist.md`: QA checklist for launching the brand across web, social, ads, and collateral.
