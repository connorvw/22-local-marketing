# 22 Local Marketing Website System

## Website Personality

The website should feel like a serious operating system for local growth, not a decorative agency portfolio.

The design should be clean, confident, direct, and conversion-focused.

## Core Pages

- Home
- Services
- Website Builds
- Local SEO
- Paid Acquisition
- Case Studies
- About
- Contact / Book a Call
- Local Audit landing page

## Hero Pattern

Use:

- strong literal headline
- short supporting copy
- primary CTA and secondary proof link
- restrained orange frame detail
- proof point or KPI below the fold

Example headline:

Home services marketing built for growth.

Example support:

We build the website, local SEO, and paid acquisition systems that turn searches into qualified calls.

## Components

### Navigation

- Bone background
- Onyx wordmark
- simple text links
- dark CTA button

### Service Cards

- White background
- Onyx border
- 8 px max radius
- Blaze eyebrow
- short description

### Proof Blocks

- KPI number in large Sora
- supporting context below
- use Slate for explanation
- use Blaze for positive emphasis

### Forms

- White fields
- Onyx or Border Strong outlines
- Blaze focus state
- direct labels
- no decorative form wrappers

### CTAs

- Blaze button for primary action
- Onyx button for high contrast
- White button with Onyx border for secondary action

## Layout Rules

- Use full-width sections with constrained content.
- Avoid card-inside-card layouts.
- Keep sections scannable.
- Use one main thought per section.
- Put proof close to claims.
- Make phone/contact actions easy to find.

## Content Modules

Recommended modules:

- Hero
- Trust bar
- Problem section
- Service system
- KPI proof
- Case study preview
- Process
- FAQ
- Final CTA

## Mobile Rules

- Keep headlines large but not cramped.
- Preserve 24-32 px side padding.
- Stack cards cleanly.
- Keep CTA buttons full-width only when useful.
- Make phone number and booking CTA easy to tap.
