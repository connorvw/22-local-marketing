22 Local Marketing complete no-margin logo exports

This folder contains all common raster formats generated from the final SVG logos.

Formats included:
- PNG: best for transparent logos and general design use.
- JPG: flattened on white for platforms that do not support transparency.
- WebP: modern web format with good compression and transparency support.
- AVIF: modern web format with strong compression and transparency support.

Sizing:
- Logo lockups: 256w, 512w, 1024w, 2048w.
- Icon/mark assets: 16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024.
- Each format also includes a tight source export at natural trimmed dimensions.

Notes:
- PNG, WebP, and AVIF preserve transparency where applicable.
- JPG files are flattened on white.
- Wordmark SVG source files use Sora converted to vector paths, so raster exports do not depend on installed fonts.
